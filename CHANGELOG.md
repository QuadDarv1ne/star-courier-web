- 🛤️ **Система Путей** (Альянс/Наблюдатель/Независимость)
- 🎭 **3 уникальных финала** (Изгнание, Договор, Слияние)
- 🧠 **Ментальное состояние** (отслеживание влияния Сущности)
- 📚 **26 персонажей** с романтическими линиями
- 📜 **40+ квестов** для глав 6-18
- ⚡ **50+ способностей** уровней 50-100
- 🌌 **18 глав** в 4 арках

### API
- `GET /api/game-mechanics/state` - состояние игрока
- `POST /api/game-mechanics/resonance/gain` - опыт Резонанса
- `POST /api/game-mechanics/path/choose` - выбор пути
- `POST /api/game-mechanics/ending/check` - проверка финала
- `POST /api/game-mechanics/entity/contact` - контакт с Сущностью

### Изменено

## [2.0.0] - 2025-12-01

### Добавлено
- 🚀 **FastAPI backend** с полной поддержкой async
- 🎨 **Vue.js 3 frontend** с Composition API
- 🔐 **JWT аутентификация** (access + refresh токены)
- 📊 **Система достижений** (22 достижения в 8 категориях)
- 🏆 **Таблица лидеров**
- 📈 **Аналитика** (отслеживание событий, статистика, воронки)
- 💾 **Экспорт/импорт данных** (GDPR compliance)
- 🔔 **Real-time уведомления** через WebSocket
- 📧 **Email уведомления** (приветствие, достижения, восстановление)
- 🛡 **Безопасность** (Rate Limiting, Security Headers, CORS)
- 📦 **PWA** (офлайн поддержка, Service Worker)

### Backend
- FastAPI 0.115.6
- SQLAlchemy 2.0 с async поддержкой
- Alembic миграции
- PyJWT для аутентификации
- bcrypt для хэширования паролей
- middleware: Rate Limiting, Security, Performance, Request Logger

### Frontend
- Vue.js 3 с Composition API
- Pinia для state management
- Vite для сборки
- Vue Router для навигации
- Axios для API запросов
- PWA с Service Worker

### API Endpoints
- `/api/auth` - Аутентификация
- `/api/game` - Игра
- `/api/characters` - Персонажи
- `/api/scenes` - Сцены
- `/api/leaderboard` - Лидеры
- `/api/achievements` - Достижения
- `/api/analytics` - Аналитика
- `/api/admin` - Администрирование
- `/api/data` - Экспорт/импорт
- `/api/abilities` - Способности
- `/api/quests` - Квесты
- `/api/game-mechanics` - Игровые механики

### Infrastructure
- Docker контейнеризация
- nginx reverse proxy
- SQLite (по умолчанию) / PostgreSQL
- Redis (опционально для кэширования)

## Типы изменений

- **Added** - для новых функций
- **Changed** - для изменений в существующем функционале
- **Deprecated** - для скорого удаления функций
- **Removed** - для удалённого функционала
- **Fixed** - для исправления багов
- **Security** - для изменений в безопасности

## Вёрсии

- **MAJOR** (3.0.0) - крупные изменения с breaking changes
- **MINOR** (2.1.0) - новая функциональность без breaking changes
- **PATCH** (2.0.1) - исправления багов без новой функциональности
