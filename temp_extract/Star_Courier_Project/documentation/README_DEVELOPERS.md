# Star Courier: Документация для разработчиков

## Обзор проекта

**Star Courier** — интерактивная текстовая RPG в жанре космической научной фантастики. Игрок берёт на себя роль Курьера, капитана космического корабля «Странник», который обнаруживает древнюю угрозу — Сущность, способную уничтожить всю галактику.

### Репозитории
- **Native версия (Python)**: https://github.com/QuadDarv1ne/Star-Courier
- **Web версия (Vue.js + Python backend)**: https://github.com/QuadDarv1ne/star-courier-web

## Интеграция с существующим кодом

### Структура проекта (Native версия)

```
Star-Courier/
├── characters.py      # Данные персонажей
├── dialogues.py       # Система диалогов
├── quests.py          # Система квестов
├── abilities.py       # Система способностей
├── game.py           # Основной игровой цикл
└── utils.py          # Вспомогательные функции
```

### Интеграция диалогов

```python
# В dialogues.py добавить:
from star_courier_dialogues_ch6_10 import DIALOGUES_CHAPTERS_6_10
from star_courier_dialogues_ch11_18 import DIALOGUES_CHAPTERS_11_18

ALL_DIALOGUES = {
    **EXISTING_DIALOGUES,  # Существующие диалоги
    **DIALOGUES_CHAPTERS_6_10,
    **DIALOGUES_CHAPTERS_11_18
}
```

### Интеграция квестов

```python
# В quests.py добавить:
from star_courier_quests_ch6_10 import QUESTS_CHAPTERS_6_10
from star_courier_quests_ch11_18 import QUESTS_CHAPTERS_11_18

ALL_QUESTS = {
    **EXISTING_QUESTS,
    **QUESTS_CHAPTERS_6_10,
    **QUESTS_CHAPTERS_11_18

```

## Новые игровые механики

### 1. Система Резонанса
- Уровни 1-4
- Разблокируется по мере развития Psychic
- Необходима для финала Слияния

### 2. Система Путей
- Альянс (флот, ресурсы)
- Наблюдатель (знания, Psychic)
- Независимость (свобода, сеть агентов)

### 3. Система Концовок
- Изгнание (жертва, победа)
- Договор (компромисс, Хранительство)
- Слияние (трансценденция, требует Psychic 90+)

### 4. Ментальное состояние
- Отслеживание влияния Сущности
- Влияет на доступные выборы

## Финалы игры

### Изгнание
```
Требования: Любой путь
Описание: Уничтожение якоря Сущности
Цена: Жертва (Курьер или соратник)
Результат: Галактика спасена, Курьер связан с местом победы
```

### Договор
```
Требования: Psychic 70+ ИЛИ Empathy 80+
Описание: Перенастройка станции для соглашения
Цена: Вечное Хранительство
Результат: Ограниченное сосуществование
```

### Слияние
```
Требования: Psychic 90+ + Resonance Level 4
Описание: Интеграция Сущности
Цена: Потеря части человечности
Результат: Трансцендентная эволюция
```

## Рекомендации по разработке

### Приоритеты интеграции
1. Диалоги и квесты глав 6-10
2. Система путей (глава 13)
3. Продвинутые способности
4. Система финалов

### Тестирование
- Проверить все ветки диалогов
- Убедиться в работоспособности 3 финалов
- Протестировать романтические концовки
- Проверить интеграцию с существующим кодом

### Баланс
- Psychic путь открывает больше возможностей
- Romance требует инвестиций с главы 1
- Path выбор влияет на сложность финала
