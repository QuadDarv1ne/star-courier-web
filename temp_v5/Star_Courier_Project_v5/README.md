## 🆕 Новое в версии 5.0: Разветвлённые диалоги глав 1-5

### Особенности системы диалогов:

**Множественные выборы:**
- Каждый диалог имеет 3-4 варианта ответа
- Выборы влияют на отношения с персонажами
- Некоторые выборы открывают уникальные ветки

**Система последствий:**
```python
CHOICE_CONSEQUENCES = {
    "chapter_1": {
        "pirate_negotiation": {
            "effects": {"selena_relationship": 5, "crew_trust": -5},
            "future_options": ["selena_negotiation_available"]
        },
        "pirate_battle": {
            "effects": {"selena_relationship": -20, "crew_trust": 10},
            "future_options": ["selena_enemy_permanent"]
        }
    }
}
```

### Глава 1: Пробуждение

**Сюжет:**
- Пробуждение из криосна на корабле «Элея»
- Возможная амнезия главного героя
- Первая встреча с экипажем: Рина, Алия, Надежда
- Пиратская атака — первый бой или побег

**Ключевые выборы:**
| Ситуация | Выбор | Последствия |
|----------|-------|-------------|
| Криосон | «Кто я?» | Начало с амнезией |
| Пираты | Бой | Селена — враг навсегда |
| Пираты | Побег | Селена — временное перемирие |
| Пираты | Переговоры | Возможность союза с Селеной |
| Команда | Довериться | +доверие экипажа |
| Команда | Командовать | +лидерство |

**Уникальные ветки:**
- **Путь силы** — победить пиратов в бою
- **Путь хитрости** — сбежать через астероиды
- **Путь дипломатии** — договориться с Селеной

### Глава 2: Первый контракт

**Сюжет:**
- Загадочный контейнер в трюме
- Артефакт Хранителей — первый Ключ
- Связь с артефактом — дар или проклятие?
- Возможность восстановления памяти

**Ключевые выборы:**
| Ситуация | Выбор | Последствия |
|----------|-------|-------------|
| Артефакт | Открыть | Связь с Ключом |
| Артефакт | Изучить | Медленный прогресс |
| Алия | Помочь ей | Романтическая линия |
| Алия | Память | Ритуал восстановления |

**Романтические возможности:**
- Первая романтическая сцена с Алиёй
- Восстановление памяти через ритуал азари

### Глава 3: Столкновение

**Сюжет:**
- Космическая станция «Нексус»
- Поиск информации о Ключах
- Встреча с Эйзен — таинственной информаторшей
- Раскрытие организации «Наследие»

**Ключевые выборы:**
| Ситуация | Выбор | Последствия |
|----------|-------|-------------|
| Эйзен | Нанять | Новый член экипажа |
| Эйзен | Отказать | Потеря информации |
| Бар | Слушать | Информация о Селене |
| Бар | Атаковать | Бой на станции |

### Глава 4: Тайны корабля

**Сюжет:**
- Прибытие к храму Хранителей
- Встреча с Еленой Соколовой — старой знакомой
- Раскрытие предателя на «Искателе»
- Проникновение в храм

**Ключевые выборы:**
| Ситуация | Выбор | Последствия |
|----------|-------|-------------|
| Елена | Союз | Путь Альянса |
| Елена | Отказ | Путь Независимости |
| Предатель | Раскрыть | Доверие Елены |
| Храм | Штурм | Бой с пиратами |

### Глава 5: Выбор пути

**Сюжет:**
- Активация карты семи Ключей
- Противостояние с Селеной Ро
- Выбор пути развития
- Начало гонки за Ключами

**Три пути:**

🛡️ **Альянс** — Защита галактики вместе с Альянсом
- Бонус: военная поддержка
- Союзник: Елена Соколова
- Особенность: доступ к ресурсам Альянса

👁️ **Наблюдатель** — Баланс между фракциями
- Бонус: нейтралитет
- Союзник: Эйзен
- Особенность: доступ ко всем фракциям

⚡ **Независимость** — Свобода выбора
- Бонус: автономия
- Союзник: никто
- Особенность: никаких обязательств
