| Романтические | 9 | За развитие отношений с персонажами |
| Боевые | 5 | За достижения в бою |

## 4. Интеграция всех систем

### Основной игровой цикл

```python
# В начале каждой сцены
def process_turn(game_state):
    # 1. Проверка случайных событий
    event = get_random_event(get_location_type(), game_state)
    if event:
        return event
    
    # 2. Проверка романтических событий
    romance_event = get_available_romance_events(game_state)
    if romance_event:
        return romance_event[0]
    
    # 3. Проверка достижений
    for ach_id in get_all_achievements():
        if check_achievement_unlock(ach_id, game_state):
            unlock_achievement(ach_id, game_state)
    
    return None
```

## Заключение

Дополнительные системы значительно расширяют вариативность геймплея:
- **Достижения** дают цели и награды
- **Романтические линии** добавляют эмоциональную глубину
- **Случайные события** обеспечивают реиграбельность

Все системы полностью интегрируются с существующим кодом и не требуют изменения основной структуры игры.
